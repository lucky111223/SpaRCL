# SpaRCL

Implementation for "SpaRCL: Robust Spatially-Aware Graph Contrastive Learning for Multi-Slice Spatial Transcriptomics Integration".

## Requirements
python, pytorch, torch_geometric, scanpy, anndata, numpy, pandas, scipy, scikit-learn

## Compared Algorithms
- [Harmony](https://github.com/immunogenomics/harmony)
- [PASTE](https://github.com/raphael-group/paste)
- [scTM](https://github.com/JinmiaoChenLab/scTM)
- [SpaCross](https://github.com/wenwenmin/SpaCross)
- [STAligner](https://github.com/zhoux85/STAligner)
